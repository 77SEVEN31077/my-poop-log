# Poop Log Tracker

A lightweight, privacy-first browser tool for logging bowel movement times and reviewing the interval between entries.

Live site: https://mypooplog.com/

## What it does

Poop Log Tracker is a simple web-based bowel movement log. It lets users record the time of a bowel movement and see how much time has passed since the previous entry.

The project is intentionally minimal: no account system, no cloud sync, no subscription flow, and no unnecessary interface complexity.

## Why this exists

Bowel habits are easy to misremember. A small timestamp log can help users notice their own baseline, especially during diet changes, travel, illness recovery, hydration changes, medication adjustments, or periods of stress.

The tool is not designed to diagnose anything. It is a lightweight personal awareness tool.

## Features

- Log bowel movement time
- Show records in a simple table
- Calculate the interval from the previous entry
- Store log data locally in the user's browser
- No account required
- No cloud database for bowel movement records
- Includes informational guide pages about tracking, privacy, and when to seek medical advice
- Mobile-friendly layout
- Simple static-site structure

## Privacy model

Bowel movement records are stored in the browser using localStorage. The website does not require a login, email address, or user account.

Because the data is local to the browser, it can be deleted if the user clears browser data, switches browsers, uses private browsing, or changes devices.

The site may use Google Analytics for basic page-view analytics and Google AdSense for advertising if approved. User-entered bowel movement log data should not be sent to analytics tools.

## Medical disclaimer

Poop Log Tracker is for personal awareness only. It is not a medical device, diagnostic system, treatment tool, or emergency service.

Users should consult a qualified healthcare professional if they notice blood in stool, black tarry stool, persistent abdominal pain, unexplained weight loss, ongoing constipation or diarrhea, or a sudden change in bowel habits.
